# AGENTS.md

## Scope

This directory contains project documentation for Eclipse OT site customization.

## Documentation Guidelines

- Keep docs practical and operational.
- Update documentation when code, SQL, deployment, configuration or security behavior changes.
- Prefer examples that match this repository's paths and current deployment model.
- Do not document secrets, real credentials, real player/account data or private infrastructure details.
- Keep command examples copy-paste friendly and clearly label destructive or production-impacting operations.

## Important Files

- `introduction.md`: project overview.
- `requirements.md`: server/runtime requirements.
- `install.md`: installation flow.
- `configuration.md`: MyAAC and theme configuration.
- `theme.md`: theme customization guide.
- `sql-migrations.md`: database migration guidance.
- `operations.md`: deploy, maintenance and rollback.
- `security.md`: hardening and secret handling.
- `changelog.md`: project change history.

## Style

- Match the existing Portuguese documentation style.
- Keep headings descriptive.
- Add checklists for operational procedures when useful.
